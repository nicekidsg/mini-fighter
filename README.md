# ⚔ 小斗士 Mini Fighter

网页版四人乱斗小游戏，玩法致敬经典横版格斗（如 LF2 的纵深走位 + 技能乱斗 + 击飞翻滚），
但**所有代码与美术均为原创**（Canvas 骨骼动画小人 + 程序化人设画像），不包含任何原游戏素材。

- 🎮 本地单机：你 VS 3 个电脑（随机职业）
- 🌏 跨地区联机：最多 4 人同房间乱斗（自带 WebSocket 服务器，角色选择互相可见）
- ⌨️ 一键放技能，无需搓招
- 🥋 十大职业，各有专属普攻与 3 个专属技能

## 快速开始

```bash
cd lf-web
npm install
npm start
```

打开 http://localhost:3000

> 只玩单机的话，直接双击 `public/index.html` 也可以（联机功能需要服务器）。

## 操作

| 按键 | 功能 |
|---|---|
| `W A S D` / 方向键 | 移动（上下是纵深走位） |
| `K` / 空格 | 跳跃 |
| `J` | 普攻（武器/职业各不相同，可空中使用） |
| `L`（按住） | 防御（伤害降为 25%） |
| `U` `I` `O` | 角色专属技能（消耗 MP，部分有冷却） |
| `M` | 静音 |
| `Esc` | 回到菜单 |

MP 自动恢复。被击倒后有短暂无敌；死亡 3 秒后复活，计入 K/D。

## 跨地区联机

1. 把这个目录部署到任意支持 Node.js + WebSocket 的平台（Render / Railway / Fly.io / 自己的 VPS 都行），启动命令 `npm start`，平台会注入 `PORT` 环境变量，已自动适配。
2. 朋友们打开同一个部署后的网址，输入**相同的房间号**点"联机对战"即可，最多 4 人。
3. 局域网内也可以：一人 `npm start`，其他人访问 `http://<这台电脑的IP>:3000`。

### 联机原理（给好奇的人）

每个客户端模拟自己的角色与飞行道具，以约 15Hz 通过服务器中转广播状态；
命中判定由攻击方发起、受击方结算（victim-authoritative），其余客户端对远程角色做位置插值。
这种结构对延迟宽容，适合休闲乱斗，但不防作弊——和朋友玩没问题。
